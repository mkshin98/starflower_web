우주 여행

밤하늘의 멋진 명소를 우주 여행하듯 둘러 보자. (작업 중)

초기 화면: 은하의 모습, 나선팔 확인
우리 은하 마을 (국부은하군) 안드로메다 여행
처녀 초은하단 여행

따라하기

  1. 왼쪽 볼거리 목록에서 사투르누트 성운 선택. 오른쪽 화면에서 파란색 선 확인 (너무 가까워서 잘 안 보임)
  2. 목록 아래의 '성도에서 찾기(Go To)' 버튼 클릭. 우측 화면에서 성운의 위치를 확인하고, 중앙 화면에서 멋진 사진을 감상한다.
  3. 좀 더 자세한 설명이 궁금하면, '설명 보기(인터넷)'을 클릭한다. (누군가 설명 작성 중이다 ^^)

  4. LMC (대마제란성운) 선택.
  5. '성도에서 찾기(Go To)' 버튼 클릭
  6. 상단 도구바에서 '3D' 버튼 클릭. 우측 마우스 클릭 후 위쪽으로 많이 드래그하여 LMC가 은하 평면 아래쪽에 위치하는 것을 확인한다. 은하 평면 (격자선)이 작게 보이도록 하면 보기 좋다.
  7. 상단 도구바에서 축소(Zoom Out) / 확대(Zoom In) 버튼 클릭하여 보기 편한 비율을 선택한다.

  8. 화면을 계속 축소해가며, 안드로메다 및 각종 은하단을 감상하자.
  9. 처녀 은하단이 길게 늘어진 모습도 확인하자.
  10. 처녀 은하단 주위에 각종 은하단이 호위병처럼 전후좌우에 펼쳐진 모습을 살펴 보자. 3D 시점을 좌우로 이동하면 작은 은하 그룹이 더 잘 보인다.

우주 여행에 좀 더 도움되는 설명

  • 맨 좌측 화면은 우주의 멋진 볼거리 목록을 보여 주며, 화면을 좀 더 넓히면 거리 정보를 확인할 수 있다.
  • 중앙 화면은 은하좌표로 보이는 성도이다. 즉, 지구에서 은하수를 수평으로 바라볼 때의 모습이다.
  • 우측 화면은, 1차로 은하의 평면도를 볼 수 있고, 여기서 나선팔의 모습, 지구의 위치 등을 알 수 있다.
  • 우측화면 상단 맨 우측 '3D' 버튼을 클릭하면 3차원 모드로 전환된다.
  • 왼쪽 마우스를 클릭하여 이동하면 성도의 중심이 움직이고,
  • 3D 모드에서 오른쪽 마우스를 클릭하여 이동하면 3D 시점이 변경된다.
  • 맨 좌측에서 대상을 클릭하면, 대상의 위치 또는 방향이 우측 화면에 파란색 선으로 표시된다.
  • 대상이 선택된 상태에서 '성도에서 찾기(Go To)' 버튼을 누르면, 먼저 우측 평면도에서 위치를 찾아 주고, 다음으로 중앙 화면에서 대상을 찾아간다. 이때, 대상의 사진이 있으면 함께 보여준다.
